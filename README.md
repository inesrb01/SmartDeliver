# SmartDeliver

## Description
SmartDeliver est une application web de gestion de livraison locale destinée aux petites
entreprises telles que les restaurants, pharmacies et commerces de proximité.

L’application permet de gérer les commandes, d’assigner les livraisons et de suivre
l’état des commandes à travers différents rôles utilisateurs.

## Type d’application
Application Web – C# (ASP.NET)

## Utilisateurs cibles
- Client
- Commerçant
- Livreur
- Administrateur

## Objectif du projet
Concevoir et implémenter un système d’information structuré, maintenable et évolutif,
en appliquant les bonnes pratiques d’ingénierie logicielle et d’architecture.
